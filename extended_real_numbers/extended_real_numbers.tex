\stitle{Extended Real Numbers}

\ssection{Why}

That some limits grow
without bound leads us
to add two elements
to the set of real numbers.

\ssection{Definition}

The set of
\ct{extended real numbers}{}
is the union of
the set of real numbers
with a set containing
two distinct elements:
one which we call
\ct{positive infinity}{}
and one which we call
\ct{negative infinitty}{}.
