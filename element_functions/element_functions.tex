\stitle{Element Functions}

\ssection{Why}

Take an element
of an algebra,
and consider the
function defined on
the ground set
which maps elements
to the result of the
operation applied to
the fixed element and
the given element.

\ssection{Definition}

Let $(A, +)$ be an algebra.
For each $a \in A$,
denote by $+_a: A \to A$
the function defined
by
\[
  +_a(b) = a + b.
\]
We call $+_a$ the
\ct{left element function}{leftelementfunction}
of $a$.

Similarly, denote
by $+^a: A \to A$ the function
defined by
\[
  +^{a}(b) = b + a.
\]
We call $+^a$ the
\ct{right element function}{rightelementfunction}
of $a$
