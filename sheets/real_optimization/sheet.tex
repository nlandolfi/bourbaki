\ssection{Why}

We discuss optimization
in which the ordered set
is the set of real numbers.

\ssection{Definition}

Let $A$ be a non-empty set and
let $f: A \to \R$.
In the context of
real optimiation, we call $f$
the \ct{objective}{} function.
