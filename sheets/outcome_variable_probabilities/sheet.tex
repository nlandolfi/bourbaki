\ssubsection{Random variable probabilities}
The \t{probability} that a random variable $x: \Omega  \to \R $ \t{equals} $a$, written $x = a$, is defined $\mathbfsf{P} (\Set{\omega  \in \Omega }{x(\omega ) = a})$.
