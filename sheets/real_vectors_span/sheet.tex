\section*{Definition}

The \t{span} of a set of vectors $v_1, \dots , v_k \in \R ^n$ is the set
\[
\span(\set{v_1, \dots , v_k}) = \Set{\alpha _1v_1 + \cdots + \alpha _kv_k}{\alpha _i \in \R },
\]
the set of all linear combinations of the vectors $v_1, \dots , v_k$.
