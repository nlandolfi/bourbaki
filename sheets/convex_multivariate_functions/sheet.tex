\section*{Why}

We generalize convex functions to $\R ^n$.

\section*{Definition}

Let $A$ be a convex subset of $\R ^n$.
The function $f: A \to \R $ is \t{convex} if for any $a, b \in A$ and $t \in [0, 1]$,
    \[
f(ta + (1-t)b) \leq tf(a) + (1-t)f(b).
    \]
It is \t{concave} if $-f$ is convex.
