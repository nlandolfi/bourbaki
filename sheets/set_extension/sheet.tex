\ssection{Why}

When are two
sets the same?

\ssection{Definition}

Two sets are
the same if and only
if they have the same
elements.
\ct{equal}{setequal}
if and only if
they have the same
elements.

\ssubsection{Notation}

Let $A$ and $B$ be
two sets.
As with any objects,
we denote that
$A$ and $B$
are equal
by $A = B$.
