\ssection{Why}

When talking about cases which
may or may not happen, we need
to enumerate or concieve of the
set of possibilities.

\ssection{Definition}

A \ct{(finite) set of outcomes}{}
is a finite set.
We call an element of the
set of outcomes an \ct{outcome}{}.
