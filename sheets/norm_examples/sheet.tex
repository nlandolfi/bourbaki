We give some standard examples

\ssection{Examples}

\begin{expl}
  The absolute value
  function is a norm
  on the vector space
  of real numbers.
\end{expl}

\begin{expl}
  The Euclidean distance
  is a norm on the various
  real spaces.
\end{expl}
