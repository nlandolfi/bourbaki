
\section*{Definition}

The \t{generated moonotone class} for a set of subsets is the smallest monotone class containing the set of subsets.
We must prove the existence and uniqueness of this monotone class

\begin{proposition}

\label{intersectmonotoneclass}The intersection of a non-empty set of monotone classes on the same base set is a monotone class.
\end{proposition}
