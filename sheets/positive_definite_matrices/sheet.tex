\ssection{Definition}

A matrix $A$ is \ct{positive definite}{}
if all quadratic forms with the
same vector on each side are postive.

\ssubsection{Notation}

Let $A \in \R^{d \times d}$. $A$
is positive definite if for every
$x \in \R^d$, $x^T A x > 0$.
