\ssection{Definition}

A \ct{probability density function}{}
of a real-valued random variable is
a measurable function from
the real numbers to
real numbers which is
(a) non-negative and
(b) the integral
of the function over a
cover-length-measurable set
with respect to the cover
measure to the cover length
is
the probability that the
random variable takes
value in that set.
We often abbreviate
probability density
function as \ct{p.d.f}{pdf},
read aloud as \say{pdf}.

\ssubsection{Notation}

Let $(X, \SA, \mu)$ be a
probability space.
Let $f$ be a real-valued
random variable on $X$.
Let $\lambda$ denote
the cover length.
A function $g: R \to R$
is a probability density
of $f$
if $g \geq 0$ and
of $f$ if
\[
  \mu(\Set*{x \in X}{f(x) \in A}) = \int_{A} g d\lambda.
\]
