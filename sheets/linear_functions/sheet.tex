\section*{Definition}

A function $f: \R ^n \to \R $ is \t{linear} if
  \begin{enumerate}
  \item $f(x + y) = f(x) + f(y)$ for all $x, y \in \R ^n$ and
  \item $f(\alpha  x) = \alpha  f(x)$ for all $x \in \R ^n$ and $\alpha  \in \R $.
  \end{enumerate}
There are simple consequences to these conditions.
For example, $f(0) = 0$.
