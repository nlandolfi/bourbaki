\section*{Definition}

Let $X$ a finite nonempty set.
A function $f: \powerset{X} \to \R $ is \t{submodular} if
\[
f(S \cup T) + f(S \cap T) \leq f(S) + f(T)
\]
for all $S, T \subset X$.
