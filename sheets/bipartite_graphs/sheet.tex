\ssection{Definition}

A \t{bipartite graph} is a triple $(A, B, E)$ so that $A \intersect B = \emptyset$ and $(A \union B, E)$ is an undirected graph.\footnote{There is a directed variant.}

\begin{proposition}
  A bipartite graph has no odd cycles.
\end{proposition}
