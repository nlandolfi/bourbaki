2. Definition

The \t{line through} two points $x$ and $y$ in $\R^n$ is the set $$ \Set*{z \in \R^n}{z = (1-a) x + y \text{ for } a \in \R \text{ and } x, y \in \R^n}. $$ An \t{affine} set is a subset of $\R^n$ that contains the line through any two of its points.

2.1 Other Terminology

Some authors call affine sets \t{affine varieties} or \t{linear varieties}.

3. Examples

The empty set is is an affine set. So is $\R^n$. \begin{prop} All singletons are affine sets. \end{prop}
