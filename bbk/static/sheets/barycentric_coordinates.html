2. Defining Result

\begin{prop} If $M = \aff\set{b_0, b_1, \dots, b_m}$ then for each $x \in M$ there exists $(\lambda_i)$ such that $$ x = \lambda_0 b_0 + \lambda_1 b_1 + \cdots + \lambda_m b_m $$ with $\sum_{i} \lambda_i = 1$. The $(\lambda_i)$ are unique if the set of vectors is affinely independent. \end{prop}

The \t{barycentric coordinates} for a vector $x$ in the affine hull of a set of affinely independent vectors is the sequence of unique coefficients expressing the vector as an affine combination of the set of vectors.
