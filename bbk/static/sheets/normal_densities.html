1. Why

We want a density that is symmetric about some central value with some spread.

2. Definition

A normal density is one which can be written as a normalized

Let $f: \R \to \R$ be a density. If there exists $\mu \in \R$ and $\sigma \in \R$ with $\sigma > 0$ so that for each $x \in \R$ \[ f(x) = \normaldensity{x}{\mu}{\sigma} \] then $f$ is a \t{normal density}. Some call a normal density a \t{Gaussian density}. We often drop the word density and use refer to these as \t{normals} or \t{Gaussians}, using these words as substantives.

We call the special case when $\mu = 0$ and $\sigma = 1$ the \t{standard normal density}.
