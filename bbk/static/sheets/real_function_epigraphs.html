\section*{Definition}

Let $f: D \to \R $ be a multivariate real-valued function where $D \subset \R ^d$. The \t{graph} of $f$ is the set in $\R ^{d + 1}$ defined by \[ \Set{(x, f(x)) \in \R ^d \times \R }{x \in D}. \] The \t{epigraph} of $f$ is the set in $\R ^{d + 1}$ defined by \[ \Set{(x, t) \in \R ^{d}}{x \in D \text{ and } f(x) \leq t}. \]

\section*{Notation}

We denote the epigraph of a function $f$ by $\epi f$.

\subsection*{Connecting convex sets and convex functions}

A function $f: \R ^d \to \R $ is convex if and only if $\epi f$ is a convex set. \subsection*{Note on terminology}

The prefix "epi} is Greek, meaning \say{above".
