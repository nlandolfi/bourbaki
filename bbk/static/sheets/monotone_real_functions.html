1. Why

We can interpret a real function as tracing a path as we move from left to right in its domain.\footnote{Future editions will likely have this interpretation in a separate sheet.} We want language for whether this tracing increases or decreases the range values.

2. Definition

Let $A \subset \R$ and let $f: A \to \R$. A function is \t{monotone increasing} if $f(x) < f(y)$ whenever $x < y$, and \t{monotone nondecreasing} if $f(x) \leq f(y)$ whenever $x, y \in \R$ and $x < y$.\footnote{Unforunately, some authors use "monotone increasing} for \say{monotone nondecreasing} and use the terminolgy \t{strictly monotone increasing} for \say{monotone increasing}." Similarly we define \t{monotone decreasing} and \t{monotone nonincreasing}.
