1. Definition

(provisional)

An \t{transformation} is a function between vector spaces over the same field. A transformation is just a function with a particular domain and codomain.

The utility of this definition is that in place of saying "a function between two vector spaces with the same field} we can \say{a transformation". So when we define properties of such functions between two vector spaces the properties are specific to such functions, and the dependence on the domain and codomain being vector spaces is more explicit.
