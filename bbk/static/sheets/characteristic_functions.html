1. Why

We want to indicate membership in a set by a function. \ifhmode\unskip\fi\footnote{ Future editions will elaborate, perhaps with forward-looking connections to \sheetref{rectangular_functions}{Rectangular Functions}. }

2. Definition

The \t{characteristic function} of a set $X$ is a function from $X$ to $2$ which is $1$ if the argument is in $A$ and 0 otherwise.

The function which assigns to each subset $A$ of $X$ to characteristic function of $A$ is a one-to-one function from $\powerset{X}$ onto $2^{X}$.

2.1 Notation

Let $A$ be a non-empty set and $B \subset A$. We denote the characteristic function of $B$ in $A$ by $\chi _{B}: A \to R$. The Greek letter $\chi $ is a mnemonic for "characteristic".

The subscript indicates the set on which the function is one. In other words, for all $B \subset A$, $\chi _{B}^{-1}(\set{1}) = B$. \ifhmode\unskip\fi\footnote{ Another notation used, when referring to these as "indicator functions,} is $1_{B}: A \to \set{0, 1}$ or $\mathbf{1"_B$. }
