2. Definition

A \t{linear functional} on a vector space $V$ with field $k$ is a function $f: V \to k$ which satisfies \[f(\lambda x + \mu y) = \lambda f(x) + \mu f(y)\] for all $\lambda, \mu \in k$ and $x, y \in V$.

2.1 Examples

\begin{enumerate} \item Define $F: \C^n \to \C$ by \[ F(x_1, \dots, x_n) = c_1x_1 + c_2x_2 + \cdots + c_nx_n \] where $c_1, \dots, c_n \in \C$. \item Let $(c_n)_{n \in \N} \in \ell^{\infty}$. Define $F_c: \ell^1 \to \C$ by \[ F_c((x_n)_{n \in \N}) = \sum_{n = 1}^{\infty} c_n x_n\]. \end{enumerate}
