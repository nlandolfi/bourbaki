2. Definition

An outcome variable is a function from a set of outcomes to a second set. We call the second set the set of values of the random variable.

3. Notation

Let $p$ be a probability mass function over the set of outcomes $A$. Let $V$ be a non-empty set. Let $f : A \to V$. Then $f$ is an outcome variable.
