2. Definition

We want to allocate a resource among $n$ entities to maximize some measure of "return} or \say{profit."

Let $B \in \R$. Let $R_i: \R \to \R$ be a function which gives the return for allocating entity $i$ the amount $x_i$ of the resource. Let $\CX = \Set*{x \in \R^n}{x \geq 0, \sum_{i}x_i = B}$ Define $f: \R^n \to \R$ by $f(x) = \sum_{i = 1}^{n} R_i(x_i)$. We call the optimization problem $(\CX, f)$ a \t{single-resource allocation problem}. In this case we call $x \in \R^{n}$ an \t{allocation} and we call $B$ the \t{budget}.
