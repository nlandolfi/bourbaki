2. Examples

Consider an integrable function defined on some measurable space. The extended-real-valued funciton which assigns to each distinguished set the value of the integrating the function over that set is a signed measure. See Example~\ref{expl:signedintegral}.

\begin{expl} \label{expl:signedintegral} Let $(X, \SA, \mu)$ a measure space. Let $R$ denote the set of real numbers. Let $f: X \to R$ an integrable function. Define $\nu: \SA \to R$ by \[ \nu(A) = \int_{A} f d\mu. \] Then $\nu$ is a signed measure. \begin{proof} First, \[ \nu(\emptyset) = \int f\chi_{\emptyset} d\mu = \int 0 d\mu = 0. \] Next, let $\seq{A}$ disjoint.
