1. Why

We divide a set into disjoint subsets whose union is the whole set. In this way we can handle each subset of the main set individually, and so handle the entire set piece by piece.

2. Definition

A \t{partition} of a set $X$ is a set of pairwise disjoint (see \sheetref{set_decompositions}{Set Decompositions}) subsets of $X$ whose union is $X$. We call the elements ofa partition the \t{pieces} of the partition. When speaking of a partition, we commonly call the set of sets \t{mutually exclusive} (by which we mean that they are pairwise disjoint) and \t{collectively exhaustive} (by which we mean that their union is full set).

2.1 Notation

Let $X$ be a set and $\CC$ be a set of subsets of $X$. $\CC$ is a partition of $X$ means $(\forall A)(\forall B)((A \in \CC \land A \in \CC) \implies A \intersect B = \emptyset)$ and $\bigcup \CC = X$.
