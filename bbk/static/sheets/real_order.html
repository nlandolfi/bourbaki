1. Why

We want to order the real numbers.\footnote{Future editions will expand}

2. Definition

For $R, S \in \R$ define the total order $\succeq$ by $R \succeq S$ if and only if $R \subset S$. As is usual with comparisons, we use the terms t{less than} and \t{less than or equal to}.

2.1 Notation

If $R$ is less than $S$ we write $R < S$. If $R$ is less than or equal to $S$ we write $R \leq S$.
