2. Definition

Let $Z$ and $X$ be non-empty finite sets. We want to model a distribution $p^\theta: Z \times X \to \R$. We parameterize a family of distributions by a parameter $\theta$. We have a dataset $(x^1, \dots, x^n)$. Given a parameter $\theta^0$, we want to solve \[ \maximization{\theta}{\sum_{k = 1}^{n} \Expect_{p^{\theta^0}_{z \mid x}(z, x^k)} \left[ \log p^{\theta}(z, x) \right] } \]
