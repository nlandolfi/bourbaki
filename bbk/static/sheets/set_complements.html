1. Why

We want to consider the elements of one set which are not contained in another set. Does such a set exist?

2. Definition

Yes: use the axiom of specification on the first set with the condition that the element not be in the second set. The axiom of extension guarantees uniqueness. And so we call this set the \t{relative complement} of the latter set in the first set. We also call it the \t{difference} between the former and the latter.

2.1 Notation

Let $A$ and $B$ be sets. We denote the difference of $A$ with $B$ by $A - B$. We express $A - B$ as $$ \Set*{a \in A}{x \not\in B}. $$
