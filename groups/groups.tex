\stitle{Groups}

\ssection{Why}

We generalize the algebraic structure of
addition over the integers.

\ssection{Definition}

A \ct{group}{group} is an
\rt{algebra}{algebra} with:

\begin{enumerate}
  \item
  an associative operation,

  \item
  an identity element, and

  \item
  an inverse for each element.
\end{enumerate}

A \ct{commutative group}{commutativegroup}
is a group whose operation commutes.
