Set-Builder Notation

Why

We specify a subset of a known set via a property.

Definition

Let $A$ be a nonempty set. We use the notation \[ \Set*{a \in A}{\text{------}\;} \] to indicate a subset of $A$ that satisfies some property specified after the $\mid$. We read the symbol $\mid$ aloud as "such that." We read the whole notation aloud as "a in A such that..."

We call the notation set-builder notation. Set-builder notation avoids enumerating elements.

Example

For example, let $L$ be the set of Latin letters and $V$ the set of Latin vowels. A first notation for $V$ is $\set{\text{a},\text{e},\text{i},\text{o},\text{u}}$. A second notation for $V$ is $\Set{l \in L}{l \text{ is a vowel}}$. We may prefer the second, in cases when it saves time. This notation is really indispensable for sets which have many members, too many to reasonably write down.
