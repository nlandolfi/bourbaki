Subset System

Why

We speak of a \rt{set}{set} and a \rt{set}{set} of its \rt{subsets}{subset} satisfying properties. The utility of this abstract concept is proved by its examples, in future sheets.

Definition

A subset system is a a pair of sets: the second set contains subsets of the first.

We call the first \rt{set}{set} the base set. If the \rt{base set}{baseset} is finite, we call the subset system a finite subset system. A distinguished subset is an element of the second set. An undistingished subset is a subset of the first set which is not \rt{distinguished}{distinguishedsubset}.

Notation

Let $A$ be a set and $\mathcal{A} \subset \powerset{A}$. We denote the subset system of $A$ and $\mathcal{A}$ by $\tuple{A, \mathcal{A}}$, read aloud as \say{A, script A.}

Example

\begin{expl} Let $A$ be a nonempty set. Let $\mathcal{A}$ be $\powerset{A}$. Then $(A, \mathcal{A})$ is a subset system. \end{expl}
