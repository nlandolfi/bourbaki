\ssection{Definition}

A
\ct{random variable}{randomvariable}
is a real-valued measurable function
on a probability space.

\ssubsection{Notation}

Let $(\Omega, \mathcal{A}, P)$
be a probability space.
Let $R$ denote the set
of real numbers.
Then $x: \Omega \to R$
is a random variable.
