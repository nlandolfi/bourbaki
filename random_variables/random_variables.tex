\ssection{Definition}


A
\ct{random variable}{randomvariable}
is a measurable map
from a probability space
to a measurable space.
Often, the codomain
is the real numbers.
In this case we say
the random variable
is a
\ct{
  real-valued random variable.
}{realvaluedrandomvariable}

\ssubsection{Notation}

Let $(\Omega, \SA,\pmeas)$
be a probability space.
Let $R$ denote the set
of real numbers.
Then $x: \Omega \to R$
is a random variable.
